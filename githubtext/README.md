# Smart Alert System

## Description
Smart Alert System is a Python Flask-based web application that helps users create, edit, and manage time-based alerts.  
The system includes user authentication and a dashboard to manage alerts efficiently.

## Technologies Used
- Python
- Flask
- HTML
- CSS
- SQLite
- Git & GitHub

## Features
- User Registration
- User Login
- Dashboard for alert management
- Add alerts
- Edit alerts
- Delete alerts
- Time-based alert system

## How to Run the Project

1. Clone the repository:
git clone https://github.com/iamrithika31-ai/githup_text2.git

2. Navigate to the project folder:
   cd githup_text2

3. Install required packages:
   pip install flask

4. Run the application:
   python app.py

5. Open your browser and visit:
   http://127.0.0.1:5000/
