
# Task Manager – Flask Web Application

##  Project Overview

This is a simple Task Manager web application** built using Python Flask and HTML/CSS.  
It allows users to add, edit, mark as completed, and delete tasks.  
The app also shows the count of pending and completed tasks.

This project is suitable for:
- Understanding CRUD operations


##  Features

- Add new tasks
- Edit existing tasks
- Mark tasks as completed / pending
- Delete tasks
- Display pending and completed task count
- Clean UI with HTML & CSS



##  Technologies Used

- Python 3
- Flask
- HTML5
- CSS3

##  Installation & Setup

## Install Flask
pip install flask

## Run the Application
python app.py
